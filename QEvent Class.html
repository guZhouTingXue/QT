QEVent Class

QEvent 类是 所有事件类的基类。事件对象包含事件参数。更多。。。


Property Documentation

Accepted:bool

事件对象的接收标志

设置接收参数指示事件接收者想要事件。不想要的事件可能不会被传播到父控件。默认下，isAccepted()被设置为true，但是 不要信任它，因为子类可能选择清除它，在他们的构造体内。

为了方便，接收标志同样可以被设置通过accept()，清除 通过 ignore().

获取函数：

Bool isAccepted()const

Void setAccepted(bool accepted)

Void QEvent::accept()

设置事件对象的接收标志，对等的调用是 setAccepted（true）。

设置接收参数 指示事件接收者想要这个事件。不想要的事件可能被传播到父控件。

参见：ignore（）

Void QEvent::ignore()

清除事件对象的接收标志参数，它的等价调用为setAccepted(false).

清除接收标志指示 事件接收者不想要这个事件。不想要的事件可能不会被传播到父控件。
